Choose How To Find A Student

This screen allows you to select the means by which to Find a Student. There are two options.

The first option allows you to search for the student using their unique Student Reference Number. The Student Reference Number is the one allocated to the student when they submitted their main grant funding.

The second option allows you to search for the student using a combination of their first name and last name, as well as an optional date of birth.
