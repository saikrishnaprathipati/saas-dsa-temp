Before you start

Make sure you have the information we need to complete this application. You can save your progress and come back it at any time. We will ask you:

  • for information about the student’s disabilities
  • to confirm the student’s course details
  • for the student’s recommended allowances including consumable items, equipment, software and training, Non-Medical Personal Help (NMPH) and travel expenses
  • for the estimated time and cost to carry out and create a Needs Assessment Report
